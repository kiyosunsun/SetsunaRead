# 拟物阅读器风格（暗桌面摄影风，昼夜双主题）设计规格

日期：2026-05-21

## 目标

为 SetsunaRead 统一一套「暗桌面摄影风」的视觉语言：桌面（背景）+ 书本（阅读页）为强记忆点；其它 UI（书架/面板/按钮）保持现代简洁，不做全局拟物，避免厚重与廉价感。需要昼夜双主题都同等精致。

成功标准：
- 阅读页第一眼像“真实书本放在桌面上”，但交互与信息层级仍清晰。
- 书架与面板不抢戏：现代、轻量、与阅读页风格一致。
- 任意屏幕尺寸下：阅读区居中，阴影/纹理不过度放大。

## 设计原则

1. **拟物集中**：拟物主要发生在阅读页（纸张、边缘、折角、书签、页面阴影）；控制面板采用现代半透明卡片。
2. **低饱和 + 低对比背景**：桌面永远不纯黑/纯白，使用“带微纹理的灰”。
3. **可读性优先**：任何装饰不得影响文字对比与行距节奏；装饰纹理强度要可调（tokens）。
4. **昼夜双主题独立调参**：不是简单反色；夜间强调“桌面暗、纸张暖、墨色柔”，日间强调“桌面浅灰、纸张更白、阴影更轻”。

## 设计 Tokens（核心变量）

### 颜色（建议用 CSS 变量实现）

#### Dark（夜间）
- `--desk-bg`: #0B0E12（桌面基底）
- `--desk-grain`: rgba(255,255,255,0.035)（桌面颗粒/噪点叠加）
- `--paper-bg`: #E7DDC9（纸面）
- `--paper-edge`: #D2C2A6（纸边旧化/书页边缘）
- `--ink-primary`: rgba(20,18,14,0.92)（正文墨色）
- `--ink-secondary`: rgba(20,18,14,0.62)（次要信息）
- `--ornament`: rgba(20,18,14,0.35)（角花/装饰线）
- `--ui-surface`: rgba(20,20,20,0.62)（面板/工具条底）
- `--ui-border`: rgba(255,255,255,0.12)
- `--ui-text`: rgba(255,255,255,0.88)
- `--accent`: #8B5CF6（蝴蝶紫/强调色，少量使用）

#### Light（日间）
- `--desk-bg`: #E6E8EC（桌面浅灰）
- `--desk-grain`: rgba(0,0,0,0.03)
- `--paper-bg`: #F7F1E6（纸面更白但保留暖）
- `--paper-edge`: #E6D7BE
- `--ink-primary`: rgba(22,18,12,0.9)
- `--ink-secondary`: rgba(22,18,12,0.55)
- `--ornament`: rgba(22,18,12,0.28)
- `--ui-surface`: rgba(255,255,255,0.72)
- `--ui-border`: rgba(0,0,0,0.10)
- `--ui-text`: rgba(0,0,0,0.72)
- `--accent`: #7C3AED

### 形状与间距
- 圆角：
  - `--radius-sm`: 10px（按钮/小卡片）
  - `--radius-md`: 14px（面板）
  - 阅读页不做圆角，保持纸张直角（更像真实书页）。
- 间距：以 8px 为基准；工具条高度与面板内边距至少 16px。

### 阴影与层级
- 桌面→书本：大而软的环境阴影（不要强方向性）
  - `--shadow-book`: 0 18px 60px rgba(0,0,0,0.45)（dark）/ 0 14px 40px rgba(0,0,0,0.18)（light）
- 纸张内阴影（页面弧度/纸张厚度）：
  - `--shadow-paper-inner`: inset 0 0 22px rgba(0,0,0,0.10)（dark）/ inset 0 0 18px rgba(0,0,0,0.08)（light）
- 面板浮层：比书本弱一档（不抢主视觉）
  - `--shadow-ui`: 0 10px 24px rgba(0,0,0,0.28)（dark）/ 0 10px 22px rgba(0,0,0,0.14)（light）

### 材质强度（可调节）
- `--grain-opacity`: 0.35（桌面颗粒强度，dark/light 分别调）
- `--paper-texture-opacity`: 0.18（纸纹强度）
- `--edge-wear-opacity`: 0.22（纸边旧化强度）

## 关键组件规范

### 1) 阅读区域（书本容器）
- 书本始终居中，周围留出桌面可见边框（至少 24px）。
- 书本阴影固定，不随滚动/翻页剧烈变化。
- 阅读页的背景（纸张）必须与文字对比稳定：纹理永远在文字之下且不超过 `--paper-texture-opacity`。

### 2) 页面（单页/双页/滚动）
- 纸张：纯色底 + 轻纸纹 + 轻内阴影形成“纸面深度”。
- 页面边缘旧化仅存在于外沿 8–12px 区域，不侵入正文排版区。
- 装饰角花（filigree）使用 `--ornament`，放在页面四角，透明度低。

### 3) 折角（狗耳）
- 折角出现在右页右上角：折线清晰、干净；折角下方投影自然。
- 折角覆盖区域的文字应可“流入折角下方”（视觉上允许被遮挡），但不可造成不可逆的信息丢失：折角区域内的文字不应承担必须信息（如章节标题/页码）。

### 4) 书签
- 书签为近黑竖条，顶部白色绳圈。
- 书签文字使用无衬线（系统 UI 字体）以形成材质对比；正文仍使用衬线/宋体类。
- 紫蝴蝶作为唯一强彩色元素，用于形成记忆点，但面积严格受控。

### 5) 工具条与面板（现代化，轻拟物）
- 采用半透明毛玻璃/磨砂卡片：`--ui-surface` + 边框 `--ui-border` + `--shadow-ui`。
- 按钮采用低对比 hover：通过亮度/透明度变化，而不是强颜色跳变。
- 所有面板视觉层级必须低于书本：不覆盖书本主体时更好（可贴边/底部）。

## 字体与排版建议

- 正文：优先系统衬线（含中文宋体/明体方向），保证长文阅读的笔画细节。
- UI：Inter/系统无衬线。
- 字重：正文 Regular；标题 SemiBold；UI Medium。

## 动效

- 原则：动效“慢、软、低幅度”，贴合摄影氛围。
- hover：缩放不超过 1.02；阴影变化不超过 15%。
- 面板进出：200–240ms，ease-out，透明度 + 位移 6–10px。

## 非目标（明确不做）

- 不做真实 3D 翻页物理模拟（成本大、性能风险高）。
- 不在全局 UI 里堆叠皮革/金属拟物材质（会变重）。
